# 🚀 Next-Gen 3D Portfolio

A futuristic, high-performance developer portfolio built with **React + Vite**, powered by **Three.js (WebGL)** and **Framer Motion** for smooth animations and immersive 3D experiences.

---

## ✨ Features

* 🌐 **3D WebGL Experience** using Three.js
* 🎨 **Modern UI/UX** with TailwindCSS
* ⚡ **Lightning Fast** with Vite
* 🎥 **Smooth Animations** using Framer Motion
* 🧠 **Interactive Components** (hover, scroll, parallax)
* 🌌 **Custom Shaders & Effects** (optional GLSL support)
* 📱 **Responsive Design** for all devices
* 🧩 Modular and scalable architecture

---

## 🛠️ Tech Stack

* **Frontend**: React 18 + Vite
* **3D Engine**: Three.js
* **Animations**: Framer Motion
* **Styling**: TailwindCSS
* **Icons**: Lucide React
* **Math Utilities**: Maath

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Abhishek-DS-ML-Gupta/NextGenPortfolio.git
cd NextGenPortfolio
```

Install dependencies:

```bash
npm install
```

---

## 🚀 Running the Project

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## 📁 Project Structure

```
mythreejs/
│── public/
│── src/
│   ├── assets/        # Images, models, textures
│   ├── components/    # Reusable UI + 3D components
│   ├── scenes/        # Three.js scenes & logic
│   ├── shaders/       # GLSL shaders (optional)
│   ├── pages/         # Portfolio sections (Home, About, Projects)
│   ├── hooks/         # Custom React hooks
│   ├── App.jsx
│   ├── main.jsx
│── index.html
│── package.json
│── tailwind.config.js
│── vite.config.js
```
